# Code of Conduct

## Our Pledge
We pledge to make participation in this project a respectful, welcoming, and harassment-free experience for everyone.

This project is focused on thoughtful engineering, constructive discussion, and shared learning.

## Expected Behavior
- Be respectful and considerate in communication.
- Assume good intent; ask clarifying questions before escalating.
- Provide constructive feedback focused on the work, not individuals.
- Respect differing viewpoints and experience levels.

## Unacceptable Behavior
- Harassment, discrimination, or personal attacks.
- Trolling, insulting comments, or deliberate disruption.
- Publishing private or sensitive information without consent.
- Any behavior that would reasonably make others feel unsafe or unwelcome.

## Scope
This Code of Conduct applies to:
- GitHub issues and pull requests
- Code reviews and discussions
- Project-related communication channels

## Enforcement
Project maintainers are responsible for clarifying and enforcing this Code of Conduct.
They may remove, edit, or reject contributions or comments that violate these guidelines.

## Reporting
If you experience or witness unacceptable behavior, please report it privately to the project maintainers.

---
This Code of Conduct is intentionally lightweight and adapted for a technical, open-source collaboration context.
